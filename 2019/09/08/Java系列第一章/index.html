Java的编译环境（win10）

一、JDK的下载安装

​ 可以选择甲骨文官方下载https://www.oracle.com/cn/downloads/或者华为云开源镜像站下载

https://mirrors.huaweicloud.com/openjdk/ 选择下载你需要的版本。

​ 下载完成后傻瓜安装，就很奈斯。

二、路径配置

​ 最后一步也是最重要的一步，路径配置/很多人也许不理解为什么要进行路径配置，简单的说，如果把Java当成是你的老婆，那么路径配置的目的，就是让你无论在何时何地，都能随意在哪里召唤出你的老婆 /：

​ 1、此电脑（右键）— 属性 — 高级系统变量 — 环境变量 — 系统变量（新建）

!

如图，变量名设为JAVA_HOME;变量值设为你JDK的安装路径，单击确定

​ 2、系统变量 — path — 编辑 — 新建

上移至第一位，单击确定。如果不出故障的话，JDK路径配置，就这么简单。

​ 3、测试

​ 召唤出万能的工具狗cmd，让小狗亲自为你嗅探，尝试召唤老婆Java

​ 召唤公式如下：（win+R）— 键入cmd — enter — 键入Java -version

​ 如果你发动了召唤公式，最终输出结果是这样的：

![](D:\Blog\blog图片\Java系列（一）\Java -version测试成功.png)

​ 那么恭喜你召唤成功，获得老婆一只，并且获得老婆的版本序列号

​ 但是，如果你召唤之后是样子的：

![](D:\Blog\blog图片\Java系列（一）\Java -version测试失败.png)

​ 那么也要恭喜你，获得一次重头开始路径配置的学习机会，（失败就是成功的开始，耐下性子重新检查你的路径配置）记住不需要重装JDK，重新配置你的路径即可。尝试几次之后，你就会干我一样，爱上这款——呸呸呸，获得一只老婆。

而接下来，我要讲述的，是正式学习Java语言之前的一些基础技巧

常用的DOS命令

1、cd 目录路径 进入一个目录
2、cd .. 进入父目录
3、dir 查看本目录下的文件和子目录列表
4、cls 清除屏幕命令
5、上下键 查看敲过的命令
6、Tab键 自动补齐命令

常用Java开发环境介绍

之前通过记事本进行java文件开发，一些有经验的同学肯定会不屑一顾：这么麻烦就为了得出一串字符，还不如直接打开文档键入输出呢。确实，之前使用记事本作为java文件开发软件时为了让大家更加具体地了解java的工作原理，所以显得繁琐而冗长，正如我们了解到的，老婆要想做好饭，就一定要有好的炊具，emmmmm，这个比喻其实不太恰当，不如这么说，要想跟老婆造娃，总不能在荒郊野岭随便开枪吧（虽然有些人确实有这样的喜好），大部分人，当然是希望有更好的环境，造娃也就更省心。那么有没有这样的一个开发环境呢？当然有！

Notepad++

UltraEdit

EditPlus

以上三款都可以升级版的记事本，如果说记事本是荒郊野岭，那么这三款可以看成是茅草房，至少可以挡风不漏雨，使用上也方便了不少，但是有些胖友就是喜欢追求极致，就是要总统套房，就是要海滨酒店，就是要落地大窗还是夜光闪闪，有没有这样一个舒适的集成开发环境呢？有！当然有！

eclipse https://www.eclipse.org/downloads/ 入门的胖友可以尝试这一款，不用序列号（免费使用）不需要各种杂七杂八的VPN翻墙下载，绝对适用于入门的胖友，功能对于我们来说也绝对够用了

IntelliJ IDE https://www.jetbrains.com/idea/（外网的东西，正常官网渠道下不了，可自己寻求资源//之后我可能单独创一期关于这一类网站及软件资源的分享,咕咕咕//而且需要序列号。但如果你喜欢网页开发，那我个人是极为推荐这一款的，页面简洁，功能强大，当然正版价格也不贵）

NetBeans https://netbeans.org/ 这一款我并没有接触，但是用过的人都说可以，见仁见智，大家用了才知道吧

为了更让大家有一个更加清晰的了解，接下来为大家奉上eclipse的下载教程，食堂也差不多开饭了，话不多说，直接切入正题：
